# **App Name**: TechFlow Mandiri

## Core Features:

- Smart Service Portfolio: An interactive showcase for all services including Windows reinstallation, hardware repair, and design services with dynamic pricing cards.
- Conversion-Optimized Booking Form: A centralized form to capture user data, service type, and WhatsApp contact information.
- Order Management System: Securely save all incoming service requests and client details to a database for business tracking.
- AI Service Assistant: An AI diagnostic tool that allows users to describe their tech problem to receive an immediate recommendation for which service package to book.
- WhatsApp Automation Bridge: A feature to automatically generate and format order messages to be sent directly to the owner's WhatsApp API.
- Business Insight Dashboard: A dedicated about section featuring core strengths and professional achievements to build client trust.
- Dynamic Map & Contact Hub: Embedded Google Maps integration with actionable contact links for real-time navigation and inquiries.

## Style Guidelines:

- Primary Color: Deep Technological Blue (#3B82F6), chosen to evoke trust and modern startup aesthetics.
- Background Color: Sleek Night Charcoal (#0F172A), providing a sophisticated dark mode look characteristic of high-end software services.
- Accent Color: Vibrant Cyan (#06B6D4), an analogous hue used for calls-to-action to signify innovation.
- Font Recommendation: 'Inter' sans-serif for its machined, neutral, and highly legible quality across all screen sizes.
- Thin-stroke tech-inspired icons from the Lucide collection to maintain a professional, minimalist startup feel.
- A responsive single-page application layout with generous white space (and dark space) to focus on service readability.
- Soft fade-in transitions for the service list and hover-state scaling for buttons to improve user engagement.
